## ハマり・発見

### @testing-library/react のクリーンアップ問題
- vitest の `globals: false` 設定では、Testing Library のオートクリーンアップが動作しない
- 既存テストは全て `afterEach(() => cleanup())` を明示的に呼んでいた
- 新テストで cleanup を忘れると DOM がテスト間で蓄積し、「multiple elements found」エラーが大量発生
- **対策**: 全テストファイルに `afterEach(() => cleanup())` を追加

### findAllByRole vs waitFor パターン
- `findAllByRole('button')` はマッチする要素が1つでも見つかった時点で返る
- 非同期ロード後に要素数が増えるケースでは `waitFor` でポーリングする必要がある
- フィルタボタン(4個) + カード群を待つ場合: `waitFor(() => expect(screen.getAllByRole('button').length).toBeGreaterThan(4))`
